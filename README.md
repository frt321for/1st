# 个人博客网站 | Personal Blog

一个现代化的个人博客网站，采用大胆的配色和炫酷的视觉效果。

## ✨ 特性

- 🎨 **大胆的渐变配色** - 使用鲜艳的紫色、粉色和蓝色渐变
- 🌟 **粒子背景效果** - 互动式粒子动画背景
- 💫 **流畅的动画** - 页面元素具有平滑的过渡和动画效果
- 🔮 **玻璃态设计** - 现代化的毛玻璃效果卡片
- ⚡ **悬停特效** - 丰富的鼠标悬停交互效果
- 📱 **响应式设计** - 完美适配各种屏幕尺寸
- 🎭 **故障艺术效果** - 标题文字的独特故障动画
- 💝 **打字机效果** - 动态文字输入动画
- 🎪 **3D 变换** - 卡片的 3D 旋转和倾斜效果

## 🎨 设计亮点

### 配色方案
- 主色调：霓虹粉 (#ff006e)
- 次色调：电子紫 (#8338ec)
- 强调色：科技蓝 (#3a86ff)
- 背景色：深空黑渐变

### 视觉效果
- 发光效果和阴影
- 渐变背景和边框
- 浮动动画
- 脉冲动画
- 涟漪效果
- 闪烁动画

1. 克隆项目到本地

## 🛠️ 技术栈

- **HTML5** - 语义化标签
- **CSS3** - 高级动画和变换
- **JavaScript** - 动态交互
- **Particles.js** - 粒子效果库
- **Google Fonts** - Poppins 字体

## 📱 响应式支持

- 桌面端 (> 1200px)
- 平板端 (768px - 1200px)
- 移动端 (< 768px)

## 🎯 主要功能模块

1. **首页** - 带有粒子背景的英雄区域
2. **关于我** - 三栏卡片展示技能
3. **博客** - 网格布局的文章卡片
4. **联系** - 表单和社交媒体链接

## 🎨 自定义指南

### 修改配色

在 `styles.css` 中的 `:root` 部分修改 CSS 变量：

```css
:root {
    --primary-color: #ff006e;
    --secondary-color: #8338ec;
    --accent-color: #3a86ff;
    ...
}
```

### 修改动画速度

调整各个 `animation` 属性的持续时间：

```css
animation: float 6s ease-in-out infinite;
```

### 修改粒子效果

在 `script.js` 的 `particlesJS` 配置中调整参数。
